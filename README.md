# ZSF Analysis 2

Modulare LaTeX-Zusammenfassung (4-spaltig, Landscape-A4) fuer Analysis 2.

## Build

```
make build          # PDF nach build/main.pdf + analysis2_fs2026_hliddal.pdf
make all            # Checks + Build
make check          # main-full, chapter-rules, root-clean, lint, pdf-identity
make release-proof  # SHA256 von build/main.pdf
make clean
```

## Struktur

Siehe [MODULAR_SYSTEM.md](MODULAR_SYSTEM.md). Layout-Anpassungen ausschliesslich in `styles/`,
nie in Kapiteln oder `preamble.tex`.

## Pruefungskontext (ETH Zuerich Maschinenbau)

Dieses PDF ist die persoenliche **Zusammenfassung der Vorlesung** (Cheat Sheet) und faellt
in diese zugelassene Hilfsmittel-Kategorie.
Zulaessig sind bis zu **10 A4 Seiten** (5 Blaetter beidseitig oder 10 Blaetter einseitig).

Die Formelsammlung **"Formeln, Tabellen, Begriffe" (FTB)** ist eine separate, zusaetzlich
zugelassene Kategorie.
